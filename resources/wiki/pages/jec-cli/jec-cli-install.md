# JEC CLI Install

## Requirements

JEC CLI needs the following system parameters in order to work correctly:

* Node 6+
* npm 3+
* TypeScript 2+

## Installation

Set up the JEC CLI module with:

```bash
$ npm install -g jec-cli
```

## Usage 

**JEC commands:**

```bash
$ jec help
```

**GlassCat commands:**

```bash
$ glasscat help
```
