# JEC CLI Install

## Requirements

JEC CLI needs the following system parameters in order to work correctly:

* Node 6+
* npm 3+
* TypeScript 2+

## Installation

Set up the JEC CLI module with:

```shell
    $ npm install -g jec-cli
```

## Usage 

**JEC commands:**

```shell
    $ jec help
```

**GlassCat commands:**

```shell
    $ glasscat help
```
